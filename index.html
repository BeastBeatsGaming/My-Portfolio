01. About Me

Hello! I'm Sai, a passionate developer who thrives at the intersection of game development and web technologies. My journey began with a fascination for creating interactive experiences that captivate and engage users.

As a game developer, I specialize in building immersive gameplay mechanics, optimizing performance, and creating worlds that players love to explore. I work with the Unity engine, bringing ideas to life through C#, and robust game architecture.

On the web development side, I craft responsive, performant applications that deliver seamless user experiences. From frontend interfaces to backend systems, I leverage modern technologies to build scalable solutions that solve real-world problems.

When I'm not coding, you'll find me exploring new game mechanics, contributing to open-source projects, or experimenting with the latest web technologies. Or maybe I'm just taking a break and enjoying playing my favorite games.
